# 📝 To-Do List App

A clean and interactive **To-Do List web application** that helps users manage daily tasks efficiently.  
Built with core web technologies and focused on simplicity, usability, and performance.

---

## ✨ Features
- ➕ Add new tasks instantly  
- ❌ Delete tasks with one click  
- ✅ Mark tasks as completed  
- 💾 Persistent task handling (session-based)  
- 📱 Responsive design for all screen sizes  

---

## 🛠 Tech Stack
- 🌐 **HTML5** — structure 
- 🎨 **CSS3** — styling  
- ⚡ **JavaScript** — logic
